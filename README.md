# Portafolio Web — Zakaria Kebour Dahmoun

Este repositorio contiene mi portafolio personal, desarrollado con el objetivo de presentar mi experiencia, proyectos y conocimientos en desarrollo web y backend.

## Tecnologías utilizadas:
- **HTML5**: estructura principal del sitio.
- **CSS3**: estilos personalizados y diseño responsivo.
- **JavaScript**: efectos visuales, animaciones y lógica de interacción.

### Recursos multimedia
- **Video de fondo** en la sección principal.
- **Imágenes e iconos** organizados en carpetas para uso dentro de la interfaz.

### Diseño y estilo
- Estilo minimalista basado en tonos oscuros.
- Tipografía: **Orbitron** desde Google Fonts.
- Efectos de sombra, animaciones con `@keyframes` y transiciones CSS.
- Elementos interactivos como cursor personalizado, botón de desplazamiento y animación de aparición gradual (reveal).

### Estructura del proyecto
- `index.html`: archivo principal del sitio.
- Carpeta `imagenes/`: iconos e imágenes del portafolio.
- Carpeta `video/`: archivo multimedia utilizado en el encabezado.
- Scripts integrados en el propio documento para controlar animaciones y eventos.
